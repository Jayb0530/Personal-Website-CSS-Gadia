Programming Language Translator

A computer program that converts the programming instructions written in human convenient form into machine language codes that the computers understand and process.

Operating System

a software program that manages a computer's hardware and software resources. It allows users to interact with their computer by running programs, storing files, and connecting to networks.

Productivity Software

A program or application that helps people create and produce content more efficiently. It can include tools for word processing, spreadsheets, presentations, databases, and more.
